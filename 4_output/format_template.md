## 格式检查点

使用此模板后，输出前必须检查：

- [ ] 所有一级标题使用 `# [数字] [标题]` 格式
- [ ] 所有二级标题使用 `## [数字.数字] [标题]` 格式
- [ ] 图例说明位于版本信息之后、第1章之前
- [ ] 每章之间用 `---` 分隔
- [ ] 表格使用 Markdown 格式，有表头和对齐符
- [ ] 所有图标（📢⚠️❗📋✅❌）使用正确
