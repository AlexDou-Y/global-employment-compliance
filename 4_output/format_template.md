# 手册格式模板骨架

## 使用说明

生成手册时，**必须先基于此模板创建骨架**，再填充内容。禁止从零构建标题结构。

骨架与 `5_spec/handbook_structure.md` 严格对齐（14 章，合同与解除合并在第 7 章，末章为法律/资料汇总）。

## 格式检查点

使用此模板后，输出前必须检查：

- [ ] 共 14 个一级标题，编号连续（无 15 章）
- [ ] 所有一级标题使用 `# [数字] [标题]` 格式
- [ ] 所有二级标题使用 `## [数字.数字] [标题]` 格式
- [ ] 图例说明位于版本信息之后、第 1 章之前
- [ ] 每章之间用 `---` 分隔
- [ ] 表格使用 Markdown 格式，有表头和对齐符
- [ ] 所有图标（📢⚠️❗📋✅❌）使用正确
- [ ] 第 7 章包含解除与遣散费（不另起一章）
- [ ] 末章为"法律/资料汇总"（不是单独的"风险章 + 资料章"）
