# 术语与表述标准

## 图例标准（固定）

**每份手册开头必须包含以下图例说明：**

```markdown
> **图例说明：** 📢 法律更新预警 ｜ ⚠️ 风险提示 ｜ ❗ 注意事项 ｜ 📋 实务要点 ｜ ✅ 合规做法 ｜ ❌ 违规/禁止
```

## 禁用词表与标准替换

| 禁用表述 | 标准替换 | 使用场景 |
|:---------|:---------|:---------|
| HRBP操作要点 | 📋 实务要点 | 操作建议段落 |
| HRBP操作清单 | 📋 操作检查清单 | 流程检查表 |
| HRBP应对建议 | 📋 实务建议 | 应对策略段落 |
| HRBP注意事项 | ❗ 注意事项 | 提醒段落 |
| HRBP操作流程 | 操作流程 | 流程标题 |
| HRBP实务操作要点 | 📋 实务操作要点 | 实务段落 |
| HRBP发出XX前检查 | 发出XX前检查 | 检查清单标题 |
| XX操作HRBP时间节点 | XX操作时间节点 | 时间表标题 |

## 图标使用规范

| 图标 | 含义 | 使用场景 | 示例 |
|:-----|:-----|:---------|:-----|
| 📢 | 法律更新预警 | 2024年后生效的新法规、即将生效的法律变更 | `📢 法律更新预警：2026年1月1日起，最低工资调整为...` |
| ⚠️ | 风险提示 | 高风险操作、常见违规行为、处罚后果 | `⚠️ 风险提示：未签订书面合同将面临双倍工资赔偿` |
| ❗ | 注意事项 | 操作细节、时限要求、特殊情况 | `❗ 注意：试用期最长不得超过6个月` |
| 📋 | 实务要点/操作清单 | 操作建议、检查清单、实务指引 | `📋 实务要点：建议在Offer中明确试用期时长` |
| ✅ | 合规做法 | 推荐的操作方式、有效的条款 | `✅ 合规做法：在劳动合同中明确约定工作地点` |
| ❌ | 违规/禁止 | 禁止的操作、无效的条款 | `❌ 禁止：不得在合同中约定"放弃加班费"` |

## 术语一致性要求

### 1. 角色称谓
- ✅ 使用："企业"、"雇主"、"员工"、"劳动者"
- ❌ 避免："公司"、"老板"、"打工人"、"HR"、"HRBP"

### 2. 文件名称
- ✅ 使用："劳动合同"、"Offer Letter"、"员工手册"
- ❌ 避免："合同书"、"录用通知"、"公司制度"

### 3. 流程描述
- ✅ 使用："操作流程"、"办理步骤"、"申请程序"
- ❌ 避免："HRBP操作"、"HR流程"、"人事办理"

### 4. 时间表述
- ✅ 使用："操作时间节点"、"关键时限"、"法定期限"
- ❌ 避免："HRBP操作时间"、"HR时间表"

## 输出前自检

生成手册后，执行以下检查：

```bash
# 检查是否包含禁用词
grep -n "HRBP" [手册文件]
```

如发现禁用词，必须全部替换后才能交付。
