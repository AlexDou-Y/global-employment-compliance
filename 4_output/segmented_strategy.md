# 分段输出策略

由于完整手册内容量大，采用分段输出：

## 输出节奏
- 每次输出 **2-3 章**
- 各批次之间**不需要等待用户确认**，直接连续输出直到手册完成
- 用户可随时打断并要求调整、补充或跳过某章节

## 推荐输出顺序
1. 第一批：1 序言 + 2 国家概况 + 3 招聘与录用
2. 第二批：4 薪酬与福利 + 5 个税 + 6 社保
3. 第三批：7 合同 + 8 假期
4. 第四批：9 工会与员工手册 + 10 数据合规
5. 第五批：11 劳动争议 + 12 外汇管制 + 13 风险提醒 + 14 资料汇总

## ⚠️ 输出完成后强制写入文件

**全部章节输出到对话后，必须立即执行以下步骤，不得跳过：**

1. 创建最终文件（如不存在）：
   - 路径：`~/.claude/skills/global-employment-compliance/[国家名称]用工合规手册_YYYYMMDD_Vx.x.md`
   - 用 Write 工具写入前50行（文件头 + 第1章）

2. 用 Edit 工具分批追加剩余章节，每批不超过50行

3. 写入完成后：
   - 用 Read 工具读取文件末尾，验证最后一章内容完整
   - 用 `wc -l` 或 Python 统计行数，告知用户文件总行数和字数

4. 告知用户文件保存路径

## 中断恢复

如输出中断：
1. 告知用户当前进度（已完成的章节编号）
2. 读取草稿文件（`[国家]_draft.md`）确认已有研究内容
3. 从断点继续，不重复已完成章节
