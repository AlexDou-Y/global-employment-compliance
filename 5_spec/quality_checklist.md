# 质量验收清单

- [ ] 每个法律规定都有"企业如何操作"说明
- [ ] 每个数字（金额、天数、比例）都有计算示例或公式
- [ ] 每个流程都有明确的步骤说明
- [ ] 每个风险点都有"如果……会怎样"场景
- [ ] 如有特殊区域，有与普通地区的对比说明
- [ ] 14 大模块完整覆盖
- [ ] 核心数据使用 **加粗** 标注
- [ ] 所有标题使用有序编号
- [ ] 必答问题表全部回答
- [ ] 社保区分本国人/外国人/双边互免
- [ ] 所有事实有检索来源支撑
- [ ] 员工手册说明强制性门槛（如有）
- [ ] 雇佣记录保存期限覆盖多角度（劳动法/税务）
- [ ] 数据合规说明跨境传输条件（如适用）
- [ ] 工具包只列有实际链接的工具
