# 输出前强制检查指南

## ⚠️ 关键原则

**禁止凭记忆输出**：必须逐章读取spec文件，对照检查后才能输出。

## 检查流程

### 1. 逐章读取spec

对每一章执行：
```
读取 5_spec/chapters/ch[XX]_*.md
```

### 2. 对照检查清单

#### 格式检查
- [ ] 章节标题层级符合spec
- [ ] 表格列数和列名与spec一致
- [ ] 使用了spec要求的标记（⚠️、📢等）

#### 内容检查
- [ ] 包含spec中的所有必答问题
- [ ] 数字、公式有具体示例
- [ ] 有计算步骤演示
- [ ] 有实务操作指南

#### 细节层级检查
- [ ] 达到spec要求的细节深度
- [ ] 有"企业如何做、何时做、由谁做"
- [ ] 有风险提示和常见错误

### 3. 缺失项处理

发现缺失：
1. 标记缺失项
2. 执行补充搜索
3. 补充完整后重新检查
4. 通过后才能输出

## 检查记录模板

```markdown
【第X章检查】
✅ 格式符合spec
✅ 必答问题全覆盖
✅ 表格完整（X个表格）
✅ 示例充分（X个计算示例）
⚠️ 缺失：[具体缺失项] → 已补充
```

## 禁止行为

❌ 跳过spec文件直接输出
❌ 凭记忆判断格式是否正确
❌ 发现缺失但不补充就输出
