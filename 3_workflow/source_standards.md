# 信息来源规范

## 优先级

| 优先级 | 来源类型 | 具体内容 |
|:------:|:---------|:---------|
| **第一** | 官方来源 | 政府官网、官方法律文本、监管机构文件、官方统计数据 |
| **第二** | 权威解读 | 顶级律所、四大会计师事务所、知名咨询公司 |
| **第三** | 参考资料 | 专业媒体（Lexology 等）、学术资源、行业报告 |

**禁止使用**：无作者百科、纯营销内容、个人博客、AI 生成内容

## 冲突处理原则

| 原则 | 要求 |
|:-----|:-----|
| **以官方来源为准** | 非官方解读不得推翻官方文本 |
| **以最新有效规则为准** | 旧法规与新法规不一致时，以最新生效版本为准 |
| **权威实务解读为补充** | 官方规则过于原则化时，可结合权威解读补充，但必须标明性质 |
| **无法确认时标注不确定性** | 存在分歧时直接说明，不得伪造确定答案 |
