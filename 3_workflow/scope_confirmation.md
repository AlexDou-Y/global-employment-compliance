# Scope 确认

## 核心规则
如目标国家存在已知的区域性法律差异（自由区、联邦州、特别行政区、自治区、省等），应主动询问企业所在具体区域。

## 常见需确认的国家（非穷举）
- UAE：Mainland / DIFC / ADGM / 其他自由区
- 中国：大陆地区 / 港澳台地区
- 美国：哪个州
- 加拿大：哪个省
- 英国：England / Scotland / Northern Ireland

## 处理方式
1. 用户明确告知区域 → 适用该区域法规
2. 用户未回复或说"不确定" → 默认全国适用，标注"需进一步确认"
3. 无区域差异的国家 → 无需询问，直接处理

⚠️ **禁止**：不要搜索"XX 国有哪些特殊区域"试图穷举。基于常识快速询问即可。
