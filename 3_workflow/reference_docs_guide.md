# 参考文档使用指南

## 使用时机
在阶段1开始前，先检查是否有用户提供的参考文档。

## 参考文档类型
- 现有手册版本（如 `UAE_HR_Manual_V3.md`）
- 法律法规文本
- 研究报告
- 官方指南

## 使用流程

### 1. 识别参考文档
检查工作目录中是否有相关文档：
- 文件名包含国家名称
- 文件名包含"手册"、"manual"、"handbook"等关键词

### 2. 读取并提取信息
- 读取文档内容
- 提取已有的法律依据、数字、公式
- 标注信息来源

### 3. 整合到研究计划
- 已有信息：直接使用，标注来源
- 缺失信息：列入研究计划
- 过时信息：标记需更新

## 参考文档 vs 网络搜索

**优先级：**
1. 参考文档中的官方法律依据
2. 网络搜索的最新官方信息
3. 参考文档中的实务解读

**注意：**
- 参考文档可能过时，需核对年份
- 关键数字需二次验证
