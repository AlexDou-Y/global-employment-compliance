# 模块间关联说明（按需参考）

| 关联模块 | 关联逻辑 | 研究要点 |
|:---------|:---------|:---------|
| **员工手册 ↔ 合同解除** | 部分国家无补偿解雇需依据员工手册惩戒条款 | 解雇条款是否需员工手册配套 |
| **雇佣记录 ↔ 税务合规** | 税务审计可能要求更长保存期限 | 区分劳动法时效 vs 税务时效 |
| **签证 ↔ 离境** | 签证注销与离境宽限期联动 | 已在签证模块整合 |
| **数据合规 ↔ 跨境派遣** | 跨国企业员工数据传输至总部需合规 | 重点关注跨境传输条件 |

⚠️ 以上关联仅供参考，应根据目标国家实际法律体系判断是否需要联动研究。
