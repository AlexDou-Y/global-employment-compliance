# 章节输出协议

## 输出原则
- 每次输出 2-3 章
- 连续输出，无需用户确认
- 每章输出前执行检查清单

## 单章输出流程

### 输出前检查
- [ ] 该章所需的研究信息已完整收集
- [ ] 信息来源已标注
- [ ] 关键数据已二次核验

### 输出格式
```markdown
## [章节号]. [章节标题]

### [子章节号] [子标题]

[正文内容]

**法律依据：** [具体法条]
**实务要点：** [操作建议]
**风险提示：** [合规风险]

---
```

### 输出后自检
- [ ] 章节结构完整
- [ ] 必答问题已覆盖
- [ ] 来源标注清晰

## 批次输出计划

**第一批（序言+基础）**
- 第1章：序言
- 第2章：国家概况

**第二批（招聘+薪酬）**
- 第3章：招聘与录用
- 第4章：薪酬与福利

**第三批（税务+社保）**
- 第5章：个税
- 第6章：社保

**第四批（合同+假期）**
- 第7章：合同
- 第8章：假期

**第五批（其他+收尾）**
- 第9章：工会与员工手册
- 第10章：数据合规
- 第11章：劳动争议
- 第12章：外汇管制
- 第13章：风险提醒
- 第14章：资源汇总
