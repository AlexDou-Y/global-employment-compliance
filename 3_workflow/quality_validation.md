# 质量验证与补全

**触发时机**：完成初版手册所有模块初稿后，**必须执行**，不得跳过。

## 质量验证三阶段

验证分为 3 个独立 pass，每个 pass 关注不同维度：

| Pass | 检查内容 | 方法 |
|:-----|:---------|:-----|
| **结构 pass** | 标题层级、编号连续性、章节齐全性 | 提取所有 `#` 行，验证编号 |
| **内容 pass** | 重复段落、信息缺口、表格完整性、链接有效性 | 逐章对照 spec |
| **实务 pass** | 可操作性、时限/金额/流程、空话段落 | 以"HR 拿到这段能不能直接执行"为标准 |

---

## 1. 结构 pass

### 检查项
- [ ] 所有一级标题使用 `# [数字] [标题]` 格式
- [ ] 所有二级标题编号连续（无跳号）
- [ ] 14 个主章节全部存在
- [ ] 图例说明位于正确位置（版本信息之后、第1章之前）
- [ ] 禁用词检查：全文无 "HRBP操作"、"HRBP注意" 等表述

### 执行方法
```bash
# 提取所有标题
grep -E "^#+ " [手册文件] | head -50

# 检查禁用词
grep -n "HRBP" [手册文件]
```

---

## 2. 内容 pass（原 HRBP 视角审查）

系统性地对初版内容提出质疑，重点关注：

| 缺口类型 | 示例 |
|:---------|:-----|
| **关联性细节缺失** | "提到了加班费，但没搜到斋月/特殊时期的工时限制" |
| **实操灰色地带** | "法律说要缴纳社保，但没说特定区域的系统怎么操作" |
| **颗粒度不足** | "只说了保存雇佣记录，没说具体保存哪些文件、保存多久" |
| **计算逻辑缺失** | "提到了离职补偿，但没说具体计算公式" |
| **程序细节模糊** | "说了要申请签证，但没提具体操作步骤或所需材料清单" |
| **例外情形未覆盖** | "只说了正常解雇流程，没说员工失联、公司破产等特殊情形处理" |

## 3. 实务 pass（原执行细节视角审查）

识别至少 5 个可能存在执行细节缺失的风险点：

| 风险类别 | 示例 |
|:---------|:-----|
| **时间边界** | 签证失效后的宽限期是几天？病假工资从第几天开始扣减？ |
| **计算逻辑** | 加班费是 1.25 倍还是 1.5 倍？年假工资按基本工资还是全额？ |
| **程序细节** | 工作许可申请需要哪些具体文件？格式要求？ |
| **系统操作** | 在哪个 Portal 提交？需要哪个部门的预审批？ |
| **例外处理** | 员工失联怎么解雇？公司破产怎么支付遣散费？ |
| **区域差异** | 不同区域的同一规定有什么不同？ |

**实务 pass 标准：** 每个段落应至少包含以下之一：
- 具体数字（金额、比例、天数、期限）
- 具体操作步骤（谁、做什么、何时、如何）
- 具体法律条款引用
- 具体风险场景和后果

如果一个段落不包含以上任何一项，考虑删除或合并。

---

## 4. 精准定向检索

针对上述问题，执行精准搜索：

```
搜索策略 = 具体法律条款号 + 目标区域 + 实操关键词

示例：
- "Federal Decree-Law 33/2021 Article 19 + overtime calculation"
- "MOHRE unpaid leave approval procedure 2024"
- "[国家] Labour Law Article X + [具体关键词]"
```

## 4. 合并更新与冲突解决

| 冲突类型 | 处理原则 |
|:---------|:---------|
| **新旧法规冲突** | 以最新生效版本为准 |
| **官方 vs 非官方冲突** | 以政府官网、官方法律文本为准 |
| **不同官方来源冲突** | 以直接主管机构口径为准 |
| **仍无法确认** | 标注"⚠️ 存在口径分歧，建议咨询本地律师" |

## 5. 质量检查清单

验证后，手册应达到以下标准：

- [ ] 每个法律规定都有"企业如何操作"说明
- [ ] 每个数字（金额、天数、比例）都有计算示例或公式
- [ ] 每个流程都有明确的步骤说明
- [ ] 每个风险点都有"如果……会怎样"场景
- [ ] 如有特殊区域，有与普通地区的对比说明
- [ ] 14 大模块完整覆盖
- [ ] 核心数据使用 **加粗** 标注
- [ ] 所有标题使用有序编号
- [ ] 重点实务问题表全部回答
- [ ] 社保区分本国人/外国人/双边互免
- [ ] 所有事实有检索来源支撑
- [ ] 员工手册说明强制性门槛（如有）
- [ ] 雇佣记录保存期限覆盖多角度（劳动法/税务）
- [ ] 数据合规说明跨境传输条件（如适用）
- [ ] 工具包只列有实际链接的工具
- [ ] 招聘合规前置要求已说明（本地优先/反歧视等，如适用）
- [ ] 劳动法覆盖范围已明确（哪些员工受/不受特定保护，如适用）
- [ ] 外籍用工附加成本已说明（Levy/配额/打分制度等，如适用）
- [ ] 裁员补偿说明资格条件和市场惯例金额（如适用）
- [ ] 数据泄露通报义务已说明（如适用）
