# 信息分发指南

## 使用时机
完成阶段1-2的全面信息收集后，将信息分配到各章节。

## 分发原则

### 1. 按主题分类
将收集的信息按主题归类到对应章节：
- 工作时间、加班 → 第4章 薪酬与福利
- 年假、病假、产假 → 第8章 假期
- 合同类型、试用期 → 第7章 合同
- 签证、工作许可 → 第3章 招聘与录用

### 2. 信息存储格式

```markdown
## [章节号]. [章节名]

### 已收集信息

#### [二级标题]
**法律依据：** [法条]
**关键内容：** [要点]
**数字/公式：** [具体数字]
**来源：** [URL]

#### [二级标题]
...

### 缺失信息
- [ ] 缺失项1
- [ ] 缺失项2
```

## 分发后检查
- [ ] 所有收集的信息已分配
- [ ] 每个章节都有对应信息
- [ ] 标注了缺失项
