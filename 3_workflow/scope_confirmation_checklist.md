# Scope 确认

## 核心确认项

### 1. 地理范围确认
- 目标国家/地区：__________
- 是否存在区域性法律差异？
  - 是 → 明确适用的州/省/特区/自由区
  - 否 → 适用全国统一法律

### 2. 章节结构确认
- **一级标题**：固定14章（见 handbook_structure.md）
- **二级标题**：根据国家特点可调整和补充
- **细节层级**：参考 spec 文件标准

## 确认输出

```
【适用范围】
国家/地区：[名称]
适用法域：[全国 / 特定区域]
特殊说明：[如DIFC等特殊区域的差异]

【章节结构】
一级标题：14章（固定）
二级标题：根据实际情况调整
```

用户确认后进入研究阶段。
