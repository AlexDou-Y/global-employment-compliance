# 重点实务问题表（必答项）

以下问题为每份手册的必答下限，可根据国家特点增加，不可减少。

| 模块 | 问题 |
|:-----|:-----|
| **薪酬与福利** | 最低薪资是否有适用范围（如不同职级、国籍、工种等） |
| | 薪资结算日期、周期和形式（如是否有薪资单）是否有强制要求 |
| | 是否有强制性奖金（如法定13薪）、调薪、补贴相关的内容 |
| | 加班费是否有上限，管理严格与否，具体如何计算，是否有豁免情形 |
| **工作与假期** | 是否存在灵活工时制度 |
| | 有什么特殊假期安排 |
| **合同事项** | 有没有必须包含的合同条款 |
| | 合同撰写是否有语言要求 |
| | 合同签订的最低类型（固定/不固定）和期限是什么 |
| | 合同续签是否是自动的 |
| **社保与税务** | 社保是否强制且分不分适用人群（本国/外国） |
| | 税收方面是否满足双边互免协议 |
| **雇佣与解雇** | 招聘流程有什么合规要求 |
| | 正式合同解除的通知期是多久，试用期呢 |
| | 解雇的补偿金适用条件是什么，具体如何计算 |
| **员工代表与劳动关系** | 是否必须成立工会等劳动代表组织 |
| | 是否必须编制员工手册 |

⚠️ 以上问题必须在手册正文中有对应内容明确回答，不可遗漏。
