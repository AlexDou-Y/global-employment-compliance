# 细节补充计划指南

## 使用时机
信息分发完成后，结合 spec 文件制定细节补充计划。

## 制定步骤

### 1. 对照 spec 文件
逐章对照 `5_spec/chapters/ch[XX]_*.md`：
- spec 要求的内容点是否已覆盖？
- 细节层级是否达到 spec 标准？

### 2. 对照必答问题
检查 `key_questions.md` 中的必答项：
- 是否全部回答？
- 回答是否具体？

### 3. 生成补充计划

```markdown
【细节补充计划】

## 第[X]章 [章节名]

### 缺失细节1
**问题：** [具体缺失什么]
**参考：** spec 文件第X条
**搜索关键词：** [针对性关键词]

### 缺失细节2
...
```

## 执行补充
按计划逐项搜索补充。
